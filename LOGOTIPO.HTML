DETECTOR DE PHISHING

El objetivo principal de esta aplicación es educar a las personas acerca del phishing para evitar el robo de datos personales. Esta aplicación nos brinda información para poder identificar un correo malicioso y contará con un detector de phishing para comprobar que el URL no contenga ningún tipo de malware, también tendrá una opción: TEST ANTIPHISHING, que pondrá a prueba nuestra capacidad para detectar ataques de phishing mediante casos hipotéticos que nosotros tendremos que resolver y que nos ayudarán a comprender cómo funcionan estas técnicas de robo.
